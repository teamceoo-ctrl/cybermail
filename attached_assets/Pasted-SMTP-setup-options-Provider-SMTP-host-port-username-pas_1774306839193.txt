Here’s the updated UI spec:

Design a dark hacker-style email campaign dashboard with neon green accents and terminal-inspired panels.

Add an SMTP Profiles page with two setup modes only:
1. Manual SMTP
2. Localhost Relay

Manual SMTP fields:
- Profile name
- SMTP host
- Port
- Encryption: None / STARTTLS / SSL-TLS
- Username
- Password
- From name
- From email
- Daily send cap
- Hourly send cap

Localhost Relay fields:
- Profile name
- Host (default: 127.0.0.1)
- Port
- Authentication toggle
- Username
- Password
- From name
- From email

Buttons:
- Test connection
- Send test email
- Save profile

Add a Reputation Monitor widget that warns if:
- outbound IP has blacklist issues
- SPF/DKIM/DMARC are missing
- bounce rate is too high
- complaint rate exceeds threshold

Add merge-tag support:
- {{first_name}}
- {{last_name}}
- {{email}}
- {{company}}
- fallback syntax like {{first_name | default:"there"}}

Style:
- black background
- green terminal glow
- monospace fonts
- left sidebar
- animated live log panel
- cyberpunk SaaS admin aesthetic
